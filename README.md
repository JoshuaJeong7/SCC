# Steel City Codes Extra Projects

Hi all! Thank you for using this Github to provide extra projects to your students! Here is a quick breakdown of these project repositories:

## What are these projects?
Essentially, they are opportunities for students to write bigger, coding-intensive, and very fun programs!
If a student wants to make a large, ambitious project from scratch, it can feel daunting to impossible, as there is no clear direction for where to start, and there are bound to be many errors if coding from scratch.
These programs help provide a head-start to students' coding, so that they can create large and fun projects while having a direction of what to code!

All projects will basically be composed of skeleton code for an overarching game or other miscellaneous project. Many of the hard or tedious parts of the code are already completed for the student, however they have to finish the rest of their code using their own programming skills in order to have a full game experience. Afterwards, they can take their program one step further and make their own additions to evolve the game into something even more special than before!

Not all programs are specifically projects or games. Some programs are formatted more like lectures or exercises/quizzes to test the students' skills. Nevertheless, I hope all of these projects achieve this goal!

## How do I use these projects?

1. Find a suitable project for your class (Intro to Python, Intermediate Python, Intro to Java, Intermediate Java, Advanced Java).
* *Unfortunately we do not have many Web Development projects, so you will have to create your own projects for a while until more are added. My recommendation is find popular websites online, and task students to format their webpage to look as close to the original as possible!*

2. Once you have a project you like, copy all of the code from Github into your code editor (Ctrl/Cmd A + Ctrl/Cmd C; then in Programiz/VSCode/Replit/etc Ctrl/Cmd V)

3. READ THE COMMENTS!
* The comments provide valuable instructions for what you are coding, how to code your particular section, and any tips. You will have a much harder time without reading those comments, so please read them for your understanding!

4. Complete all program assignments stated in the comments!

5. If you have time, upgrade your code by adding your own fun additions that will make the game feel more alive!

### Happy Coding!
